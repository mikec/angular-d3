angular-d3
==========

[Angular](http://angularjs.org/) directives for [d3](http://d3js.org/)

View the [demo site](http://mikec.github.io/angular-d3)


Setup
-----

Install [grunt](http://gruntjs.com)

    $ npm install
    $ bower install


Build
-----

    $ grunt


Demo
----

View the demo locally by browsing to demo/index.html
